iSegFormer (Liu et al., MICCAI 2022)

  • Task: Interactive 3D knee MRI segmentation
  • Encoder: Swin Transformer for global context
  • Decoder: Lightweight MLP for fast mask output
  • Workflow: Clinician labels a few slices → model refines entire volume
  • Result: >90% Dice with minimal labels
  • Trade‑off: High GPU memory for 3D
