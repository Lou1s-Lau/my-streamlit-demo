Background: Deep Learning & Medical Image Segmentation

1. What is Deep Learning?
  • Deep learning uses layered “neural networks” to learn features (edges → textures → shapes).
  • Training adjusts internal weights to minimize prediction errors on labeled data.
2. Key Architectures
  • CNN (Convolutional Neural Network): uses learnable kernels to detect local patterns; backbone of UNet for segmentation.
  • ViT (Vision Transformer): splits images into patches, uses self‐attention for global context; ideal for 3D volumes.
3. What Is Image Segmentation?

Assigns a label to every pixel (e.g., “cartilage,” “bone,” “background”), yielding precise outlines—critical for clinical planning.

4. Fully Automatic vs. Interactive Segmentation
Fully Automated
  • No user input; batch processing
  • Pros: High throughput, minimal effort
  • Cons: May missegment fuzzy or rare cases
Interactive
  • User provides clicks/scribbles for guidance
  • Pros: Combines expertise with AI, corrects edge‑cases
  • Cons: Requires a few interactions
